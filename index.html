DIVs

The DIV tag on its own doesn't do much.

Defintions

  • The DIV tag has no default styling.
  • The DIV tag defines content areas.
  • HTML is not funny.

DIV tag usage

Assigning an id to a tag allows us to style that tag (and all its nested tags) directly through CSS.

To give that id styling it must be names as a selector,along with its declaration block, and preceded by a pound sign.

ID usage

The id allows us to identify tags so that they may specifically be addressed withis CSS.
